# 📚 Choose Your Course

Um componente de seleção de curso com radio buttons estilizados, exibindo nome e preço de cada opção — desenvolvido com HTML e CSS puro.

---

## 📸 Preview

> Card branco centralizado sobre fundo com gradiente verde-azul. Título "Choose Your Course" e três opções de curso em cards cinza claro com radio button, nome do curso e preço anual.

---

## 🚀 Tecnologias

- **HTML5**
- **CSS3**

---

## 📁 Estrutura do Projeto

```
choose-your-course/
├── index.html
├── style.css
```

## ⚙️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/choose-your-course.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd choose-your-course
   ```

3. Abra o `index.html` no navegador ou via **Live Server** no VS Code:
   - Acesse em: `http://127.0.0.1:5500`

- Lista de cursos com radio buttons personalizados

## 🎨 Design

- Fundo da página: gradiente verde-ciano para azul (`#43e97b → #38f9d7 → #4facfe` aprox.)
- Card principal: branco com bordas arredondadas e sombra suave
- Opções de curso: fundo cinza claro com bordas arredondadas
- Radio button: circular, cinza quando inativo
- Tipografia: sans-serif, preta/escura

---

## 📌 Observações

- Projeto estático, sem dependências externas
- Ideal como componente de seleção em plataformas de cursos online ou e-commerce educacional
